## Números por extenso
Crie um programa que receba um número, e retorne como se lê o número em uma string. Exemplo:

```
Insira um número:
85
oitenta e cinco
```

Adicionalmente, o programa pode reconhecer unidades de medida diferentes:

```
243,00 R$

```
